**Hypolia Ruby** is our REST API written in typescript with NodeJS and [AdonisJS](https://adonisjs.com). This project is open-source, like all Hypolia concepts!

## Getting Started

### Development
```bash
$ cp .env.example .env
$ yarn install
$ yarn dev # http://localhost:3333
```

### Production
```bash
$ docker compose up
```
